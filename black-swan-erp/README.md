## Run Locally

### Prerequisites

* **Node.js** (LTS version recommended)
* npm

---

### Installation

1. Install dependencies:

   ```bash
   npm install
   ```

2. Create a `.env.local` file in the root directory and add your API key:

   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

3. Run the application:

   ```bash
   npm run dev
   ```

4. Open your browser at:

   ```
   http://localhost:3000
   ```
